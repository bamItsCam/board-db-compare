About This Site

Hey there! Welcome to our site


The original concept for a site comparing single board computers (SBC's) was inspired by www.board-db.org, a site that maintains a database of SBC's currently on the market. We wanted to create a simple tool for comparing boards for those searching for the best SBC for a project (or anyone interested in how they measure up!).

The site was developed using Bootstrap CSS and the Django framework. If you have any questions, comments, or improvement ideas, please contact us on the Contact Page!
